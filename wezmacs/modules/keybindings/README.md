# Keybindings

Core keyboard bindings for pane and tab management, selection modes, and utilities.

## Required Packages

None

## Key Bindings

**Leader**: `CMD+Space` (5 second timeout)

### Pane Management
- `LEADER+-` - Split horizontal (30% below)
- `LEADER+|` - Split vertical (25% right)
- `LEADER+z` - Toggle pane zoom
- `LEADER+p` - Pane select
- `LEADER+P` - Pane swap
- `LEADER+n` - Move pane to new tab
- `LEADER+N` - Move pane to new window

### Pane Navigation
- `CTRL+Left/Right/Up/Down` - Navigate between panes

### Pane Resizing
- `CMD+r` - Enter resize mode (then use arrow keys)
- `Escape` or `Enter` - Exit resize mode

### Tab Management
- `LEADER+t` - New tab
- `LEADER+w` - Close tab

### Selection & Utilities
- `LEADER+q` - Quick select (URLs, paths, hashes, IPs, UUIDs)
- `SHIFT+Up` - Scroll to previous prompt
- `SHIFT+Down` - Scroll to next prompt
- `LEADER+Enter` - Toggle fullscreen
- `LEADER+L` - Show debug overlay
- `SHIFT+Enter` - Send newline (Claude Code compatibility)
