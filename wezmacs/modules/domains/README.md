# Domains

Quick domain management for SSH, Docker, and Kubernetes connections.

## Required Packages

None (uses WezTerm plugin: quick_domains)

## Key Bindings

- `LEADER+t then a` - Attach to domain
- `LEADER+t then v` - Domain in vertical split
- `LEADER+t then h` - Domain in horizontal split
- `LEADER+t then Escape` - Exit domains menu
