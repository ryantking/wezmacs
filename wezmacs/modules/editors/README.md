# Editors

External code editor launchers for terminal editors and IDEs in the current working directory.

## Required Packages

None (configurable terminal_editor and ide, defaults: vim and code)

## Key Bindings

- `LEADER+e then t` - Open terminal editor in smart split
- `LEADER+e then T` - Open terminal editor in new tab
- `LEADER+e then i` - Launch IDE in current directory
- `LEADER+e then Escape` - Exit editors menu
