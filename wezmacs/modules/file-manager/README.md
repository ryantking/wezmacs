# File Manager

File management with configurable terminal file manager (default: yazi).

## Required Packages

- `yazi` - Homebrew: `brew install yazi`

## Key Bindings

- `LEADER+f then f` - File manager in smart split
- `LEADER+f then F` - File manager in new tab
- `LEADER+f then s` - File manager with sudo in smart split
- `LEADER+f then S` - File manager with sudo in new tab
- `LEADER+f then Escape` - Exit file-manager menu
