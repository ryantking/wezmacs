# Git

Git workflow integration with lazygit and diff viewing.

## Required Packages

- `lazygit` - Homebrew: `brew install lazygit`
- `git` - Homebrew: `brew install git`
- `delta` - Homebrew: `brew install git-delta` (optional, for better diff formatting)

## Key Bindings

- `LEADER+g then g` - Lazygit in smart split
- `LEADER+g then G` - Lazygit in new tab
- `LEADER+g then d` - Git diff main branch in smart split
- `LEADER+g then D` - Git diff main branch in new window
- `LEADER+g then r` - Riff in smart split (if riff feature enabled)
- `LEADER+g then Escape` - Exit git menu
