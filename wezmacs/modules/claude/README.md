# Claude

Claude Code integration with workspace management.

## Required Packages

- `claude` - Homebrew: `brew install claude` or visit https://claude.sh
- `agentctl` - Included with claude installation

## Key Bindings

- `LEADER+c then c` - Create new agentctl workspace (smart split with interactive prompt)
- `LEADER+c then C` - Create new agentctl workspace (new tab with interactive prompt)
- `LEADER+c then Space` - Switch to existing agentctl workspace
- `LEADER+c then s` - Switch to existing agentctl workspace
- `LEADER+c then d` - Delete agentctl workspace
- `LEADER+c then Escape` - Exit Claude menu
