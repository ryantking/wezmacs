# Claude

Claude Code integration with workspace management.

## Required Packages

- `claude` - Homebrew: `brew install claude` or visit https://claude.sh
- `claudectl` - Included with claude installation

## Key Bindings

- `LEADER+c then c` - Open Claude in new tab
- `LEADER+c then C` - Open Claude in new tab (alias)
- `LEADER+c then w` - Create new claudectl workspace
- `LEADER+c then s` - Switch to existing claudectl workspace
- `LEADER+c then d` - Delete claudectl workspace
- `LEADER+c then Escape` - Exit Claude menu
